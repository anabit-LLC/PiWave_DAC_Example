# PiWave_DAC_Example
Example C code for Anabit's PiWave DAC which features the Raspberry Pi RP2350, runs on VS Code programming environment. 
There are three versions of the code, one for each resolution version of the PiWave: 10 bit, 12 bit, 14 bit
These examples allow you to generate various analog signals including: DC level, Sine wave, triangle wave, square wave, two tone
It also allows you to set the output impedance of the PiWave DAC: 1Mohm or 50 ohm
